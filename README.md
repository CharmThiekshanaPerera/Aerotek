## REQUIREMENTS

### PROJECT DESCRIPTION

This is a corporate website for a commercial ventilation manufacturing company.

### PROJECT STRUCTURE

There will be only 1 component in this project.

1. Website

### WEBSITE STRUCTURE

#### HOME

See mockup design in `source` branch for reference.

#### ABOUT US

This page should displays below sub sections.

- Vision and mission
- Our story
- Accolades and affiliations
- Our team
- Our brands

#### PRODUCTS

This page should displays a list of products. Each product should have a single product page and all of them are static pages.

#### PROJECTS

This page should displays a list of projects. Each project should have a single project page and all of them are static pages.

#### CONTACT US

This page should displays below components.

- Contact details
- Contact form with mail facility
